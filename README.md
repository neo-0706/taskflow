# 📋 TaskFlow

A modern Todo List application built with **HTML**, **CSS**, and **Vanilla JavaScript**.
This project focuses on creating a clean, responsive, and visually appealing task management experience using a glassmorphism-inspired interface.

---

## ✨ Features

* ➕ Add new tasks
* ✅ Mark tasks as completed
* 🗑️ Delete individual tasks
* 🧹 Clear all tasks
* 🎨 Modern Glassmorphism UI
* 📱 Responsive layout
* ⚡ Built without any frameworks

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/your-username/taskflow.git
```

2. Open the project folder

```bash
cd taskflow
```

3. Open `index.html` in your browser.

No installation or additional dependencies are required.
